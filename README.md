# TrailBlaze AI

TrailBlaze AI is a cloud-ready, AI-powered outdoor guidance platform focused on Colorado trails. It combines Retrieval-Augmented Generation (RAG), geospatial trail data, weather intelligence, and user reviews to answer natural-language questions through a modern web interface.

## New Features (v2)
- Trail similarity search (FAISS-powered)
- Plan My Hike itinerary builder
- Side-by-side trail comparison
- Live trail conditions feed
- Elevation profile charts
- Dark mode toggle
- Share trail deep-links
- Crowd density heatmap layer
- Seeded reviews database
